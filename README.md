# Habit Hero

**Daily workout tracker & wellness logger. Log workouts, track water intake, earn points, and collect unlockable icons as you build a better you — workout after workout.**

## Features

- 5-day workout plan with estimated calories and focus tags
- Daily water intake tracker (up to 128 oz)
- Points system with daily login streaks, workout logging, and water goals
- Icon shop with 10 unlockable collectibles
- Iron Rookie icon gifted automatically on first login
- All data saved locally — no account needed

## Scoring

| Event | Points |
|---|---|
| Daily login | +5 |
| Login streak bonus | +2 × streak days (max +20/day) |
| Workout logged | +25 |
| Water goal hit (128 oz) | +10 |
| Perfect Day bonus | +15 |

## Deploy

1. Push `index.html` and `manifest.json` to the repo root
2. Settings → Pages → Deploy from `main` branch, root folder
3. Live at `https://<username>.github.io/<repo-name>/`

No build step. No dependencies. Single file app.
